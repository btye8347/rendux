About RendUX

RendUX is a declarative UI shell runtime built on FastAPI, Jinja2, and HTMX.

How it works

  1. Declare — Write a views.yaml that describes your shell, views, and routes.
  2. Register — The runtime loads the config at startup and compiles the view models.
  3. Render — Each request resolves the shell context and renders the appropriate template.
  4. Inject — Your application routes and API endpoints plug into the shell via standard FastAPI.

Stack

  • Python 3.12+
  • FastAPI — routing and dependency injection
  • Jinja2 — server-side template rendering
  • HTMX — hypermedia-driven partial swaps
  • PyYAML — view configuration
