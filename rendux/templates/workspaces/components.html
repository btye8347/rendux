Components

RendUX ships primitive templates for common UI patterns. Wire them to API data via the DataMapper.

List

Flat list of items with title, description, meta, and tags. Maps from any JSON endpoint.

primitives/list.html

Grid

Grouped grid of items with status tags. Useful for service or resource dashboards.

primitives/grid.html

Detail List

Key-value pairs for displaying structured entity data or runtime status.

primitives/detail_list.html

Surface

Controlled iframe surface with trust levels, fallback modes, and action bar.

primitives/surface.html
